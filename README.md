# EWR Departure Guide — April 27, 2026

A single-page, mobile-responsive departure guide for ANA/United flight NH 6453 from Newark Liberty International Airport (EWR) Terminal C to Tokyo Narita (NRT).

Built for two audiences: the traveler (Stephen) and the driver (Colleen), with distinct sections tailored to each person's needs.

## Features

- Complete morning timeline from wake-up to wheels-up
- Step-by-step driving directions from Blackwood, NJ to EWR Terminal C
- Terminal C interior flow diagram
- Curbside drop-off visual guide
- Driver's return route home
- "What if..." FAQ for common anxieties
- Quick reference card (screenshot-friendly)
- Countdown to departure
- Dark mode support
- Print stylesheet
- Fully offline once loaded — no external dependencies

## Deployment

Single `index.html` file. Deploy to GitHub Pages by pushing to a repo and enabling Pages from the root directory.

## Tech

- Zero dependencies — pure HTML, CSS, JavaScript
- Embedded SVG diagrams
- Mobile-first responsive design
- `prefers-color-scheme` dark mode
- Print-optimized layout
